## 요구사항
[]핵심 비즈니스 로직을 가지는 객체를 domain 패키지에 구현한다
[]UI 관련한 객체를 view 패키지에 구현한다
[]MVC 패턴 기반으로 리팩토링
[]view 패키지의 객체는 domain 패키지의 객체에 의존 가능하다
[]domain 패키지의 객체는 view 패키지의 객체에 의존하지 않는다
